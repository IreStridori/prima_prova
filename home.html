The tool

This tool allows genomicists to handle FASTA files and supports various operations related to mithochondrial DNA analysis. It is able to read a FASTA file and organise it in a user friendly manner, as a pandas dataframe. Other than this, it can also provide statistics related to the uploaded sequence and to perform motif search and pairwise alignment.


How to use it?

You simply upload your FASTA file in the dedicated section and the tool does everything for you! :)
